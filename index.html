EXIM DDI

Sistem pelacakan jadwal import & export internal — ETD, ETA, dokumen kepabeanan, dan daftar barang dalam satu tampilan.

Catatan Data

Data tersimpan permanen di database online (Supabase), bukan lagi di browser. Tombol Bulk Export mengunduh seluruh jadwal (per mode) sebagai file Excel, dan Bulk Import memuatnya kembali dari file Excel dengan format yang sama.

Info
